### SUMMARY Every time control state is updated on the Explore view, the data panel requests new data, even when the changes are known not to affect the chart data (`renderTrigger` property set to `true`). This PR changes `ExploreChartPanel` to only refresh the `queryFormData` that's used by `DataTablesPane` if the `triggerRender` prop is not `true` (is falsy when a query has been issued).
